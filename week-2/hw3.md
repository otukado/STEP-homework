# キャッシュ
$O(1)$ で実現する
- 与えられた `<URL, web ページ>` が存在するか
- ない場合はキャッシュ内で最も古いものを削除し、新たなものを追加する

## 方針
- `Hash Table` を用いれば、$O(1)$ で `<URL, web ページ>` を検索できる
- `Hash Table` で削除するには削除すべき `key` を指定する必要がある → これを $O(1)$ で求めたい
- `queue` を使えば削除すべき最古の `key` が $O(1)$ で求められる
    - 検索してあった場合にどうする？
- 双方向リストを使えば、場所がわかれば $O(1)$ で重複した場合の処理ができる

- `Hash Table` に `list` のイテレータを保存する
- `Hash Table` で検索してあった場合は、 `list` のイテレータを使って `list` の中でその要素を削除し、 `list` の末尾に追加し、 `Hash Table` のイテレータも更新する
- `Hash Table` で検索してなかった場合には、`list` の先頭の要素を削除し、`Hash Table` からも削除し、 `list` の末尾に新しい要素を追加し、 `Hash Table` にも追加する
